# GyverTwink
Гирлянда на адресных светодидоах и esp8266, управление по WiFi

## Папки
- **docs** - всякая инфа, документы, протокол связи
- **firmware** - прошивка для esp8266
- **libraries** - библиотеки для esp8266
- **processing** - исходник приложения
- **schemes** - схемы
